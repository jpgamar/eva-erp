## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## EVA Embed Environment Variables

Set these on the frontend runtime (local `.env.local` or deployment env):

```bash
EVA_APP_URL=https://app.goeva.ai
EVA_API_BASE_URL=https://api.goeva.ai/api/v1
EVA_SHARED_EMAIL=hey@goeva.ai
EVA_SHARED_PASSWORD=

# Fixed target selection (recommended)
EVA_TARGET_AGENT_ID=
EVA_TARGET_PHONE=+525629151938
EVA_TARGET_GATEWAY=openclaw-gateway-e2aac289a6
EVA_TARGET_PORT=19070

# Existing backend URL used by rewrite/auth checks
API_URL=http://127.0.0.1:4010
```

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
